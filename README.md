# Praveen Bandi — Personal Site

> Deployed at **[folium.vercel.app](https://folium.vercel.app)**

A Renaissance-inspired personal portfolio, built with vanilla HTML, CSS, and JavaScript. No frameworks, no dependencies — just craft.

## Stack

- HTML5 / CSS3 / Vanilla JS
- Google Fonts: Cinzel, IM Fell English, Spectral
- Deployed via Vercel (root: `portfolio/praveen_bandi/personal-site`)

## Local Development

```bash
# Just open in browser — no build step needed
open index.html
```

## Deploy

Pushes to `main` branch auto-deploy via Vercel GitHub integration.
